# jira-pull-request

> A GitHub App built with [Probot](https://github.com/probot/probot) that Creates automatically pull request on branch created matching a jira ticket pattern

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
